3. Vue-CLI

  • 🛠️ Vue.js 开发的标准工具

可以使用下列任一命令安装这个新的包：
npm install -g @vue/cli

运行vue --version命令来检查其版本是否正确:


如需升级全局的 Vue CLI 包，请运行：

npm update -g @vue/cli

如需卸载全局的 Vue CLI 包，请运行：

npm uninstall vue-cli -g
